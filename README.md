# CloudEagle Dropbox API Integration Assignment

This repository contains the Java implementation and testing for the CloudEagle assessment on **Dropbox Business API integration**.

## Project Overview

The goal of this assignment is to:

1. Authenticate with Dropbox using OAuth2 (Code Flow).
2. Test and document the following APIs:
   - Get Team/Organization Name
   - Get Plan Type / License
   - List All Users in the Organization
   - Fetch Sign-In Events (via custom backend endpoint)
3. Implement one API in Java to fetch data and print the results.
4. Include Postman tests and screenshots for each API.
